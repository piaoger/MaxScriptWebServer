# MaxScriptWebServer

Embedded MaxScript WebServer to remote execute Maxscript

## how to use

```
cd dist
start startmax.bat
```

## how to tweak

1. You can change listening port in runwebserver.ms

2. You can even modify C# source for more customizations.
